# Frontend concept references

Five interactive HTML directions for the frozen **Visibel — From Zero to Hero** journey:

1. Landing — outcome-first promise and one clear action.
2. Suggestions — evidence-backed actions with `Improve It` / `Do It For You` interactions.
3. Outcome — a truthful Before/After preview and the final publishing question.

Open `index.html`, choose a direction, then use the floating three-step switcher. Each concept is responsive, keyboard-navigable (`←` / `→`), supports reduced motion, and uses only local files and system fonts.

The sample content comes from `code/media-ad-coach/lib/fixtures/replay-audit.json`. AI-generated placeholder imagery remains visibly labeled **AI concept**; no screen claims that the result is deployed.

## Recommended production direction

Start with **01 Luminous Focus** for the product shell, borrow the recommendation hierarchy from **04 Editorial Proof**, and keep the outcome viewer from **03 Spatial Glass**. This creates a quiet Apple-like experience while making the evidence and transformation easy to judge in ten seconds.
